## Typescript / Express / Node / Sqlite

_Setup your node backend fast_

### Principal Scripts

```bash
# Start Production Server
  npm run start
```

```bash
# Start Development Server
  npm run dev
```

```bash
# Format source code with prettier
  npm run format
```

```bash
# Check types and compile to dist folder
  npm run build
```

##### Recomendations
